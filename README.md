# 💰 Expense Tracker

A lightweight web application to track income and expenses with a clean UI and localStorage persistence. Built using **HTML, CSS, and JavaScript**.

---

## 📖 About

Expense Tracker helps you manage your personal finances by allowing you to:

- Add transactions with a name and amount
- Choose whether each transaction is **Income** or **Expense** using toggle buttons
- Automatically calculate and display your **balance, total income, and total expenses**
- View a transaction history with the ability to delete items
- Persist data using **localStorage**, so your records remain even after refreshing the page

---

## ✨ Features

- Clean and responsive UI styled with modern CSS
- Interactive toggle buttons with clear visual highlights for Income/Expense selection
- Real-time updates to balance and transaction list
- LocalStorage integration for saving and retrieving transactions
- Simple, intuitive design for everyday use
